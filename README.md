connect-owin Samples Repository
===============================

## ASP.NET WebApi Sample

	$ cd Samples.WebApi
	$ msbuild Samples.WebApi.csproj
	$ npm install
	$ node server.js

Open http://localhost:3000/Api/Hello/World

## Nancy Sample

	$ cd Samples.Nancy
	$ msbuild Samples.Nancy.csproj
	$ npm install
	$ node server.js

Open http://localhost:3000/
